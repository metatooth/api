== Description ==
A prefect Free Business WordPress Theme. This theme comes with amazing slider, header options, WooCommerce compatibility, multiple blog layouts and many more. In addition, footer section options, sidebar options, pagination types, animation disable options, search placeholder are some added features. Furthermore, home page section options, 9+ added custom widgets, multiple sidebar area, go to top optons etc. Demo: http://canyonthemes.com/demo/business-path/ 

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
