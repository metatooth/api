A Storefront child theme designed for small WooCommerce stores / boutiques. *Merchant* features a simple, traditional design which you can customise using the settings available in the WordPress Customizer. Looking for a theme for your new WooCommerce store? Look no further than Storefront and Merchant!

== Description ==

Storefront is a robust and flexible WordPress theme, designed by WooCommerce creators WooThemes to help you make the most out of using WooCommerce to power your online store. It's available to download for free at the WordPress theme repository.

Merchant is a child theme for Storefront meaning that both themes must be installed if you want to run Merchant on your store.
